# 🎨 Editor (워크플로우 에디터) - 5분 빠른 시작

## 🚀 시작하기

### 1️⃣ 에디터 열기

```
http://localhost:3000/editor
```

**또는 Dashboard에서**:
1. Dashboard 방문
2. "📊 최근 워크플로우" 섹션
3. "+ 새 워크플로우" 버튼 클릭

---

## 🎯 기본 워크플로우 만들기 (5분)

### 단계 1: 모델 로더 노드 추가

```
1. 왼쪽 패널에서 "🤖 모델 로더" 클릭
2. 캔버스에 노드 추가됨
3. "Model ID" 필드에 모델명 입력: gpt2
4. "📥 Download Model" 클릭
```

**완료**: 모델 다운로드 진행!

---

### 단계 2: 데이터셋 노드 추가

```
1. 왼쪽 패널에서 "📊 데이터셋" 클릭
2. 캔버스에 노드 추가됨
3. 포맷 선택: CSV/JSON/Parquet
4. "📂 Upload File" 클릭
5. 파일 선택
```

**완료**: 데이터셋 업로드 진행!

---

### 단계 3: 워크플로우 저장

```
1. 헤더의 "💾 저장" 버튼 클릭
2. 워크플로우 이름 입력
3. "저장" 클릭
```

**완료**: 워크플로우 저장됨!

## 🎯 노드별 사용법

### 🤖 모델 로더

```
입력:    (없음)
출력:    Model
기능:    • HuggingFace 모델 자동 다운로드
        • 로컬 모델 로드
        • 진행률 표시
```

**사용 예시**:
```
Model ID: meta-llama/Llama-2-7b
Source:   HuggingFace
Token:    (선택사항)
```

---

### 📊 데이터셋

```
입력:    (없음)
출력:    Dataset
기능:    • CSV/JSON/Parquet 지원
        • 파일 업로드
        • 데이터 정보 표시 (행, 열, 크기)
```

**지원 포맷**:
- ✅ CSV
- ✅ JSON
- ✅ Parquet

---

### 🎓 학습 (구현 중)

```
입력:    Model, Dataset
출력:    Trained Model
기능:    • LoRA/QLoRA 지원
        • 학습 파라미터 설정
        • 진행률 실시간 표시
```

---

### 💬 Chat (구현 중)

```
입력:    Model
출력:    Response
기능:    • 모델과 대화
        • 다양한 샘플링 옵션
        • 대화 히스토리
```

---

### 🔍 RAG (구현 중)

```
입력:    Model
출력:    Search Results
기능:    • 문서 로드 및 검색
        • 유사도 기반 결과
        • 청크 관리
```

---

### 📦 GGUF 내보내기 (구현 중)

```
입력:    Model
출력:    GGUF File
기능:    • 모델 양자화
        • 다양한 압축 옵션
        • 파일 크기 최적화
```

## 💾 워크플로우 관리

### 저장하기

```
1. 하단에 수정 내용이 있으면 "💾 저장" 활성화
2. 클릭하여 모달 열기
3. 이름 + 설명 입력
4. "저장" 클릭
```

### 불러오기

```
Dashboard → 최근 워크플로우 → 워크플로우 클릭
```

### 새로 만들기

```
1. 헤더의 "📄 새로 만들기" 클릭
2. 또는 Dashboard에서 "+ 새 워크플로우" 클릭
```

---

## 🎬 워크플로우 실행

### 실행 준비

```
1. 모든 노드의 필수 파라미터 설정
2. 노드 간 연결 확인 (향후 추가)
3. "▶ 실행" 버튼 클릭
```

### 실행 중

```
• 각 노드가 순차적으로 실행
• 진행률 표시
• 우측 "📤 출력 패널"에 로그
```

### 완료

```
• 실행 시간 표시
• 결과 요약
• 에러 메시지 (있을 경우)
```

## 🔍 화면 구성 요소

### 1️⃣ 헤더 (상단)

| 버튼 | 기능 |
|------|------|
| 📄 새로 만들기 | 새 워크플로우 생성 |
| 💾 저장 | 현재 워크플로우 저장 |
| ▶ 실행 | 워크플로우 실행 |
| 📦 노드 | 노드 팔레트 표시/숨김 |
| ⚙️ 설정 | 속성 패널 표시/숨김 |
| 📤 출력 | 출력 패널 표시/숨김 |

### 2️⃣ 왼쪽 패널 (노드 팔레트)

**6가지 노드 타입**:
```
🤖 모델 로더   📊 데이터셋   🎓 학습
💬 Chat       🔍 RAG        📦 GGUF
```

**통계**:
- 현재 노드 수
- 현재 연결 수

### 3️⃣ 중앙 (캔버스)

**상호작용**:
- 마우스 휠: 줌
- Shift+드래그: 팬
- 노드 드래그: 이동
- 노드 클릭: 선택
- 우측 하단: 줌/팬 정보

### 4️⃣ 오른쪽 패널

**속성 패널**:
```
선택된 노드의 파라미터 수정
```

**출력 패널**:
```
✅ 실행 완료
⏱️ 소요 시간: XXXms
📊 실행된 노드: X
```

## ❓ FAQ

### Q: 노드를 다시 배치할 수 있나?
**A**: 네! 노드 헤더를 클릭&드래그하면 이동합니다.

### Q: 수정한 내용을 되돌릴 수 있나?
**A**: 현재는 불가능합니다. (Ctrl+Z 준비 중)

### Q: 워크플로우를 공유할 수 있나?
**A**: 추후 JSON 내보내기 기능 추가 예정입니다.

### Q: 노드를 복사할 수 있나?
**A**: 현재는 불가능합니다. (복사/붙여넣기 준비 중)

### Q: 실행 중에 멈출 수 있나?
**A**: 현재는 불가능합니다. (실행 취소 기능 준비 중)
